## 已知限制

1. **编码问题**: 在 Windows 上，包含中文的路径可能存在编码问题，建议使用英文路径或确保系统编码正确设置。

2. **CLI 路径检测**: 如果微信开发者工具安装在非标准路径，需要通过环境变量 `WEAPP_CLI_PATH` 手动指定。

3. **登录交互**: `weapp_login` 工具需要在终端中显示二维码，可能需要特定的终端环境支持。

## 建议配置

{
  "mcpServers": {
    "weapp-devtools": {
      "command": "node",
      "args": ["D:\\Projects\\WeappMCP\\dist\\index.js"],
      "env": {
        "WEAPP_PORT": "9420",
